### 2. [个人自用 rsync 文件同步常用命令](https://blog.saop.cc/p/30664)

```Bash
# 将本地 test 文件夹同步传输到远程终端上
rsync -ravzP ~/test i@i.cc:/home/i/

# 将远程 test 文件夹同步传输到本地终端上
rsync -ravzP i@i.cc:/home/i/test ~/
```
