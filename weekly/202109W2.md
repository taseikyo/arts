### 1. [golang-redis 判断 key 不存在还是读取出错](https://cloud.tencent.com/developer/article/1608750)

redis 中有一个特殊值（`redis.Ni`）用来判断读取时 key 不存在的情况：

```golang
// 判断 key 是否存在
val, err := client.Get("user:zhangyunfeiVir").Result()
if err == redis.Nil {
    fmt.Println("key2 does not exist")
} else if err != nil {
    panic(err)
} else {
    fmt.Println("读取：", val)
}
```
